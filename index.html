// security + trust

Security you can verify

Important promise: RingTail will be transparent by default. We publish source, document known risks, and keep ownership in public.

Open-source releases

Every downloadable build maps to a public Git commit. Anyone can inspect changes, audit dependencies, and rebuild locally.

Policy: source first, binaries second

No silent telemetry

RingTail will not ship background analytics without explicit opt-in. If data collection is needed for a feature, it will be documented clearly.

Default: private unless you choose otherwise

Responsible disclosure

Found a security issue? Report it privately so users can be protected before public disclosure. We will acknowledge reports and track fixes publicly.

Email: security@ringtail.peanut
